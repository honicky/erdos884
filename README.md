# A formal disproof of Erdős Problem #884 (Lean 4 / Mathlib)

**Problem** ([erdosproblems.com/884](https://www.erdosproblems.com/884)): for $n$ with divisors
$1 = d_1 < \dots < d_{\tau(n)} = n$, Erdős conjectured
$$\sum_{1\le i<j\le\tau(n)} \frac{1}{d_j-d_i} \;\ll\; 1 + \sum_{1\le i<\tau(n)} \frac{1}{d_{i+1}-d_i}
\qquad (n \to \infty).$$
Terence Tao disproved this assuming the Hardy–Littlewood prime-tuples conjecture
([note](https://terrytao.wordpress.com/wp-content/uploads/2025/09/erdos-884.pdf), Sept 2025);
Daniel Larsen then gave an **unconditional** disproof
([884.pdf](https://github.com/Larsen-Daniel/Erdos-884/blob/main/884.pdf)) via a multiscale
version of Tao's construction. This repository formalizes Larsen's proof.

## Main theorem

```lean
theorem Erdos884.erdos_884_disproof :
    ¬ (sumDivisorInvPairwiseDifference =O[Filter.atTop]
        (1 + sumDivisorInvConsecutiveDifference))
```

where the two definitions are **verbatim** those of
[google-deepmind/formal-conjectures `ErdosProblems/884.lean`](https://github.com/google-deepmind/formal-conjectures/blob/main/FormalConjectures/ErdosProblems/884.lean),
and `=O[Filter.atTop]` is exactly that file's `≪`. Together with the identity
`answer(False) ↔ Erdos884Prop ↔ False`, this resolves `erdos_884` there with answer **False**.

## Design deviations from the paper (all proof-preserving)

- Interval `[t,2t] → (t,8t]` so Mathlib's Chebyshev constants (log 2 lower / log 4 upper) suffice.
- Tao's Lemma 2.2 proved without Rolle/convexity (induction + sign argument).
- Larsen's dyadic-shell counting replaced by an elementary symmetric-function bound
  `Σ_{|u|=k−1} ∏ 1/(p−p₀) ≤ ((1+log K)/N)^{k−1}`.
- The tower `t_i = exp(t_{i−1})` replaced by a greedy recursion: each new scale is chosen
  Archimedean-large relative to the current product `n_r` and its minimum divisor log-gap `δ_r`.
- `log* t` avoided: `K_r ≈ log t_r/((r+12)·loglog t_r)` gives the harmonic divergence directly.
